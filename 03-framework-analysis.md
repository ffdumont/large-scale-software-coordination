# 3. Frameworks as Possible Responses

## Research question

How do existing frameworks attempt to solve the fundamental problems identified in section 1?

## Scope

Frameworks and approaches to analyze:

- SAFe;
- LeSS;
- Nexus;
- Scrum of Scrums;
- large-scale Kanban;
- Team Topologies;
- Flight Levels;
- FAST Agile;
- Disciplined Agile;
- Shape Up;
- Spotify Model, with explicit caveats;
- hybrid enterprise models.

## Analysis template

For each framework or approach:

1. Guiding philosophy.
2. Problems it primarily addresses.
3. Assumptions about the organization.
4. Mechanisms it uses.
5. Problems it solves well.
6. Problems it solves less well.
7. Trade-offs introduced.
8. Contexts where it tends to fit.
9. Contexts where it tends to struggle.
