# Annexe C — Matrice des preuves

## Objectif

Cette matrice relie les affirmations, les mécanismes et les sources. L’objectif est d’éviter les conclusions non étayées.

## Niveaux de preuve

| Niveau | Type | Force typique |
|---|---|---|
| A | Recherche évaluée par les pairs, théorie établie, preuves empiriques reproduites | Forte |
| B | Rapports empiriques de haute qualité, recherche longitudinale sur l'industrie | Fort à moyen |
| C | Études de cas publiques crédibles et conférences | Moyen |
| D | Documentation officielle du cadre | Utile pour l'intention, faible pour l'efficacité |
| E | Avis de praticien, marketing fournisseur, anecdote | Faible sauf corroboration |
